# Advent of Code

App version of [Readme.app.md](./README.app.md)

## Configure

```sh
yarn install
```

## How to run

With ts-node

```sh
./advent 2023 1
```

With bun (ts)

```sh
./advent-bun 2023 1
```

## Tools used for some exercises

* [Graphviz](https://graphviz.org)
