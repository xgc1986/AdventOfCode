# Advent of Code

Web version of [Readme](./README.web.md)

## Configure

```sh
yarn install
```

## How to run

With ts-node

```sh
./advent 2023 1
```

With bun (ts)

```sh
./advent-bun 2023 1
```

## Tools used for some exercises

* [Graphviz](https://graphviz.org)
